# Diet & Fitness Website

A web-based Diet & Fitness Tracking System developed using PHP, MySQL, HTML, CSS, and JavaScript.

## Features

- User Registration & Login
- Dashboard
- Diet Planning
- Workout Tracking
- BMI Calculator
- Progress Tracking
- User Profile Management

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Installation

1. Clone the repository:

```bash
git clone https://github.com/aswin3002/diet-fitness.git
```

2. Copy the project to the XAMPP `htdocs` folder.

3. Import the MySQL database.

4. Start Apache and MySQL using XAMPP.

5. Open your browser and visit:

```
http://localhost/diet_fitness_website
```
