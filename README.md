# 🥗 Diet & Fitness Website

A modern **Diet & Fitness Tracking Web Application** developed using **PHP, MySQL, HTML, CSS, and JavaScript**. This project helps users manage their diet plans, track workouts, monitor BMI, and visualize their fitness progress through an interactive dashboard.

---

## 📌 Project Overview

The Diet & Fitness Website is designed to provide users with a simple and effective platform to maintain a healthy lifestyle. Users can register, log in securely, manage their fitness goals, track daily workouts, create diet plans, and monitor overall progress.

---

- 🔐 Secure User Registration & Login

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/aswin3002/diet-fitness.git
```

### Navigate to Project

```bash
cd diet-fitness
```

### Move Project

Copy the project folder to:

```
C:\xampp\htdocs\
```

### Configure Database

- Start **Apache** and **MySQL** using XAMPP.
- Create a MySQL database.
- Import the SQL file into the database.
- Update database credentials if required.

### Run the Project

Open your browser:

```
http://localhost/diet_fitness_website/
```
